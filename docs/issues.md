## 機能追加予定

### 認証機能

- メールアドレス認証機能の追加
- パスワードリセット機能の実装

### 映画情報機能

- 映画詳細情報の拡張（ジャンル・シリーズ・制作国など）
- 監督・キャスト情報の表示
- 監督・キャストによる検索・ソート機能の追加

### レビュー機能

- ユーザー定義可能なジャンルタグ機能の実装
- 評価を表現するレーダーチャート機能の追加

### シェア機能

- レビュー内容を画像として生成し、エクスポートできる機能の実装

## 改善予定

- レビュー情報取得時のデータ構造を見直し、DTO設計の整理・最適化
- エラーページの実装によるUX向上
- フロントエンドのリファクタリング
  - コンポーネント構成や責務の見直しによる可読性・保守性の向上
  - OpenAPIを利用してAPIレスポンスの型定義を自動生成することで、型安全性を高める

## 導入予定技術

- Docker
  - 開発環境およびデプロイ環境の統一のため
- GitHub actions
  - CI/CDの仕組みを理解するために導入し、簡単な自動化から実装
- OpenAPI
  - API仕様を元に型定義を自動生成し、フロントエンドとの型の不整合を防ぐため
